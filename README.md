# Instructions
This is a personal repository containing my Arch Linux setup and configurations. Feel free to explore and use it as a reference for your own setup.

## Config Files
- General configuration files for various applications and tools, such as Sway, Neovim, Zsh, and more.
- These files are typically located in the `~/.config` directory, but some may be in the home directory or other specific locations depending on the application.

### Config Included
- Sway: My Sway configuration file, which includes my custom keybindings, workspace setup, and other settings for the tiling window manager.
- Neovim: My Neovim configuration, which includes my custom settings, plugins, and keybindings for a productive coding environment.
- Kitty: My Kitty terminal emulator configuration, which includes my custom color scheme, font settings, and other preferences for a visually appealing terminal experience.
- Mako: My Mako notification daemon configuration, which includes my custom settings for notifications and their appearance.
- Fuzzel: My Fuzzel application launcher configuration, which includes my custom settings for launching applications and managing windows.
- Waybar: My Waybar configuration, which includes my custom settings for the status bar and its modules.
- Cava: My Cava audio visualizer configuration, which includes my custom settings for visualizing audio in the terminal.
- Xremap: My Xremap configuration, which includes my custom key remapping settings for a more efficient keyboard layout.
- 

### Zsh Config
- `/.zshrc`: My Zsh configuration file, which includes my custom aliases, functions, and Antidote setup.
- `/.p10k.zsh`: My Powerlevel10k theme configuration for Zsh.
- `/.zsh_plugins.txt`: List of packages to load by Antidote, my Zsh plugin manager.
- `/.zprofile`: My Zsh profile file, which is sourced by `.zshrc` for additional configurations. Handles login and startup tasks.

## Obsidian Vault
My obsidian configurations, themes, and plugins. Place them in `.obsidian` of your Obsidian vault directory.
Plugins inluded:
- Latex Suite: Ultra fast latex snippets.
- Minimal Theme: A clean and minimal theme for Obsidian.
- Style Settings: A plugin that allows you to customize the appearance of Obsidian with CSS.

## VsCode Settings
- Settings: `~/.config/Code/User/settings.json`
- Keybindings: `~/.config/Code/User/keybindings.json`
- Snippets: `~/.config/Code/User/snippets/`

### Hsnips Snippets
My custom Hsnips snippets for VSCode is located in
`~/.config/Code/User/globalStorage/draivin.hsnips`

## Scripts
- located in `.local/bin/` for easy access from the terminal.
- Make sure to give them executable permissions using `chmod +x script_name.sh` before running them.

## Additional Notes
- Make sure to back up your existing configurations before replacing them with mine.
- Some configurations may require additional dependencies or software to work correctly.
- I use timeshift for system backups, so consider setting it up for your own system.
- Feel free to reach out if you have any questions or need assistance with the setup.
- Enjoy your Arch Linux experience!
